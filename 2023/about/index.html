关于我

一般地，你可以称呼我为：Cryovit, GoRandom 或 mojimoon。

更多名字与由来

我一向来不太会起网名，因为他人有趣的网名难以模仿，而自己无聊的想象不能保证独创性。也就是说，我很难找到一个既让我满意，又不会被人抢注的名字。

而在 Github、Google、Twitter 等大型平台上，大量的用户名被占用，我也不想使用数字后缀等 hacky 的方法。

因此，我还是干脆使用多种网名，以应对不同的场合。

已弃用的网名：

  • hiuseues (2017-2018): 我在最初接触 OI 时滚键盘的产物。当时还不懂英文的发音，我自己的读法是“嗨优色斯”。
  • wseqwq (2018): 对某个六字母、qwq 结尾的网名的模仿。时至今日，我想不起来也搜不到他的名字了。顺便一提，wseq 这四个字母在键盘上是相邻的，打起来很方便。但是，不仅不遵循英文音节规则，还只能按照字母拼读。
  • OrzAM (OrzAutomaton) (2018-2019): 2018 年暑假在洛谷群里吹水时别人给我的称号。Orz 是一个表达敬意的表情符号，AM 是 AutoMaton 的缩写，意为“自动机”。这五个字母也只能按照字母拼读。
  • HexSet → HexSeq (2019)：从此开始我在 Segoe UI Emoji 字体中搜罗表情符号，用于制作头像。这个网站的 favicon 就是我当时的头像，因为它本来就非常小，而不像其他头像一样，放到 32x32 的小图标上就完全看不清了。
  • Ittsuu (2019): 日本麻雀中「一気通貫（いっきつうかん）」的简称「一通（いっつう）」的罗马音。顺便一提，当时我完全不懂日语，以为 Ikkitsuukan 的读法是 Ikkit Suukan。「您完全没学过日语是吗？」然后被群友这样评论了。
  • ReDiv (Redivision) (2019): 在 OI 时期使用过的最后一个网名。
  • TealTen (TealTenacity → TealTerminal) (2020): 从此开始我用 mspaint 制作纯文字头像。
  • Poradis (2022): 是对 parodist (打油诗人) 的一种 parody (谐仿)。中专学历的虚拟主播不会读的单词。

仍在使用的网名：

  • Unreset (Unnreset) (2020-): Unnecessary Reset 的缩写，是我对之前如此多次更替网名的「完美主义」的自嘲。果不其然，到高三为止，我根本就没找到换网名的必要。
  • Cryovit (2022-): Cryogentic Vitriol (低温矾，即水合硫酸盐) 的缩写。源于 2022 年初时在写《明日方舟》的同人时虚构的一种蓝色材料。
  • GoRandom (2022-): 高考之后对充满随机性的人生的一种自嘲。
  • もじもん (mojimoon) (2023-): 对日语文字游戏充满热情的我怎么也想不出一个好的谐音名。在试了「雪湯（ゆきゆ）」、「完走（かんそう）した感想（かんそう）」等没人用也不够有趣的名字之后，我姑且是找到了属于自己的「文字門（もじもん）」。

博客自定义

本站使用的 Stellar 主题提供了美观的 UI 和强大的功能，你可以查阅官方文档 进行自定义。

以下列出了本站部分自定义设置，以供参考。

_config.yml

_config.yml 在 hexo init 时自动生成，只需要修改替换已有内容即可；但是末尾的 inject 部分需要手动添加。

_config.stellar.yml
  • gradient：页面左上角的搜索栏渐变色。由于这个渐变色会逐渐向左移动，所以首尾颜色必须相同，看起来才连贯。此处采用了官方推荐设置。
  • font-size：默认值为 body: 15px, code: 14px, codeblock: 13px，由于本人阅读习惯，略微增大了字号。
  • font-family：除了在此处设置，还需要在 _config.yml 中 inject。此处的设置为：Lato 作为西文字体，Noto Serif SC 作为中文字体。其他字体是为了兼容不同平台。（注意把中文主字体放在西文主字体之后，否则西文字符也会使用中文字体。）
  • comments：评论系统。可以参考官方文档中的评论插件配置，以及我的评论仓库 blog-comments 的 README 来进行配置。
