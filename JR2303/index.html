模型

以下按照我接触到的顺序列出模型。

模型下载后放在 /stable-diffusion-webui/models/Stable-diffusion 目录下（其中 /stable-diffusion-webui 是根目录）。在此目录应该有一个空文件 Put Stable Diffusion checkpoints here.txt。

以下的文件大小均为 Windows 资源管理器显示的大小，与网站上的大小可能不同。文件名后的括号内为文件的 SHA256 哈希值，可以在 Webui 左上角检查。

animefull-final-pruned.ckpt [925997e9]

  • Size: 3.97GB
  • Download: novelaileak

相对位置是 /novelaileak/stableckpt/anime-final-pruned/model.ckpt。无需下载其他任何文件（包括同目录下的 config.yaml）。

显然是知名度最高的模型。
我没怎么用过，因为这玩意我半年前就下载了，一直在硬盘里积灰，等我重新捡起来 Stable diffusion，已经变成奔着别的模型去了。
当然，NovelAI 确实下了很多功夫，只可惜没有人更新 leak，这玩意也难以满足被鲜艳色彩养叼的审美了。

anythingV3_fp16.ckpt [38c1ebe3] + VAE

  • Size: 1.98GB
  • Download: Anything V3 - Civitai

需要搭配 VAE 文件 (784 MB) 使用，否则色彩会比较灰暗。

上面的链接可以一并下载 VAE 文件（animevae.pt），不过也可以在 novelaileak 的根目录找到。

下载之后，更名为anythingV3_fp16.vae.pt(格式为 模型名.vae.pt)，和模型放在同一目录下。

只有明确指定需要 VAE 的模型才应该使用 VAE。

比 animefull-final-pruned 效果要好。不过，我也只是偶尔用用。

这就要请出一位重量级嘉宾：

momoko-e.ckpt [a2a802b2] / momoko-p.ckpt [5d916d1e]

  • Size: 1.98GB / 1.98GB
  • Download: momoko - Hugging Face

鲜艳！生动！ enough said.
另外，本篇文章中唯一的 sfw 模型是 momoko-e。
再另外，这两个模型本质上不是同一个，使用同样的 prompt 和 seed，结果肯定不同。

本来我以为关于模型的探索就到这里了，但是，当我发现本地没有一个模型能正常地识别 1boy 之后，不得不说，这个世界还是很大的。

wd-v1-3-float16.ckpt

  • Size: < 2GB
  • Download: Waifu Diffusion - Hugging Face

为什么没有 Hash 和准确大小呢？因为我早就删除了。
别用。这玩意是 final 的话，前面的 epoach 我都不敢想象。

HD-16.ckpt

  • Size: < 7GB
  • Download: Hentai Diffusion - Hugging Face

别用。虽然我知道可能是我的问题，但是它的艺术风格确实是领先人类艺术的，连 illustration 都救不回来。

HD-17.ckpt [3a0cfc1c] + VAE

  • Size: 3.58GB
  • Download: Hentai Diffusion - Hugging Face

我没见有人说过 HD 需要 VAE，但是我确实观察到色彩灰暗的现象，并且在使用 VAE 之后有所改善。

然而，由于是真的没人说过，所以也没人提供 VAE 文件。

解决方案也很简单，直接用 novelaileak 的 VAE 文件就行了。

首先，比起 HD-16 可谓是天壤之别，进步显著，至少像是个二次元模型。
其次，能够识别 1boy，比起别的已经赢太多了。
再次，也不要抱太高期望，写 1girl, 1boy，会产出大量的可能性，包括但不限于 1girl, 2girls, 1girl, 2boys……不过这和别的 prompt 以及我自己的使用方式也有关系。

杂项

标签

显然，如果你想学习 prompt 怎么写，应该去找元素法典等教程及工具网站。我在下面推荐了一些。

这里只讲一点个人经验。对，就是上面提到的 1boy 问题：

masterpiece, best quality, 1 <长修饰语> girl, (1boy:1.35), <细节>

由于使用了长魔法吟唱，girl 的细节在一定程度上会受限，但是至少能够正确地画出 1girl, 1boy 了。

设置

启动后进入 Settings 页面，在第二栏底部找到 Stable Diffusion 设置中的 Stop At last layers of CLIP model，改为 2。

PNG Info

PNG Info 可以识别 Stable Diffusion 生成图片的信息，包括 prompt、negative prompt、seed 等生成参数。

注意：Send to txt2img 会使用相同的 seed，如果你想用同样的标签生成不同的图，记得重置！
