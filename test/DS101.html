Data Science 101

Data Science 101 (DS101) is a community-driven project to provide a free, open-source and beginner-friendly introduction to data science and machine learning using Python.

Inspired by Kaggle Learn, the goal of DS101 is to provide a more structured and comprehensive introduction to data science and machine learning, with a focus on the fundamentals and practical applications.

Features

  • Free and open-source: DS101 is free and open-source, licensed under the MIT License, and always welcome contributions from the community.
  • Hands-on and practical: With free online hosted Jupyter notebooks, you can learn and practice data science and machine learning without installing anything on your computer (restrictions apply).
  • Beginner-friendly and structured: DS101 is designed for beginners, but also suitable for more experienced data scientists looking to refresh their knowledge. The content is structured into modules, each covering a specific topic, with a mix of text, code and exercises.
  • Comprehensive: DS101 covers the full data science workflow, from data collection and cleaning, to machine learning and model deployment.
  • Multi language: DS101 is currently available in English and Chinese, and we welcome contributions to translate it into other languages.

Courses

DS101 features many courses that are not very long, but are very hands-on and practical. Each course is designed to be completed in a few hours, and you can pick and choose the courses that interest you the most.

First Things First

This section prepares you for the rest of the courses. Maybe you want a fresh start, or maybe you want to review the basics. Either way, the basics are important, so check whether you are familiar with the language, libraries and math used in the courses:

  • Introduction to Computer Programming: No prior programming experience required. Learn the basics of computer programming using Python.
  • Python for Data Science: A continuation of Introduction to Computer Programming. Learn the basics of Python for data science, including data structures, functions, classes, modules, and more.
  • NumPy and Pandas: Learn how to use NumPy and Pandas, to manipulate and analyze data with ease.
  • Data Visualization: Learn how to use Matplotlib and Seaborn, to create beautiful and informative data visualizations.
  • Linear Algebra: Review the basics of linear algebra, including vectors, matrices, and linear transformations.
  • Probability and Statistics: Review the basics of probability and statistics, including random variables, distributions, hypothesis testing, and more.

Working with Data

Real-world data are not always clean and ready to use. This section covers the basics of working with data, including data collection, cleaning, and wrangling:

  • Introduction to SQL: Learn the basics of SQL, including data manipulation and querying.
  • Data Collection: Learn how to collect data from various sources, including web scraping, APIs, and databases.
  • Advanced SQL: Learn advanced SQL techniques, including window functions, common table expressions, and more.
  • Data Cleaning: Learn how to clean and prepare data for analysis, including missing data, outliers, and more.
  • Data Wrangling: Learn how to transform and reshape data, including merging, pivoting, and more.

Machine Learning

Now ready to dive into machine learning? This section covers the basics of machine learning, including supervised and unsupervised learning, model evaluation, and more:

  • Introduction to Machine Learning (+Project): Learn the basics of machine learning, including supervised and unsupervised learning, model evaluation, and more. Create your first machine learning model using Scikit-Learn.
  • Feature Engineering (+Project): Learn how to create new features from existing data, including feature selection, feature extraction, and more.
  • Machine Learning with Text (+Project): Learn how to apply machine learning to text data, including text preprocessing, feature extraction, and more.
  • Introduction to Deep Learning (+Project): Learn the basics of deep learning, including neural networks, convolutional neural networks, recurrent neural networks, and more. Create your first deep learning model using TensorFlow and Keras.
  • Computer Vision (+Project): Learn how to apply deep learning to computer vision, including image classification, object detection, and more.

One Step Further

More specific and practical topics can seem overhelming. But don’t worry, we’ve got you covered. This section covers more advanced topics, including model deployment, time series analysis, geospatial analysis, recommender systems, and more:

  • Model Deployment: Learn how to deploy machine learning models, including web applications, APIs, and more.
  • Time Series Analysis: Learn how to analyze time series data, including time series decomposition, forecasting, and more.
  • GeoSpatial Analysis: Learn how to analyze geospatial data, including geocoding, reverse geocoding, and more.
  • Recommender Systems: Learn how to build recommender systems, including content-based filtering, collaborative filtering, and more.
  • Machine Learning Explainability: Learn how to explain machine learning models, including feature importance, partial dependence plots, and more.

Beyond the Basics

Yeah, you might wonder why all the courses below are titled “Basic Guide”. Well, in practice, there are many more advanced topics, but they are often built on top of the basics. So, if you want to learn more, you should first make sure you are familiar with the basics. This section covers more advanced topics, including natural language processing, computer vision, game AI, reinforcement learning, and more:

  • Basic Guide to Game AI: Learn how to build game AI, including pathfinding, decision making, and more.
  • Basic Guide to Inforcement Learning: Learn the basics of reinforcement learning, including Markov decision processes, Q-learning, and more.
  • Basic Guide to Natural Language Processing: Learn the basics of natural language processing, including text preprocessing, word embeddings, and more.
  • Basic Guide to Computer Vision: Dive deeper into computer vision, including image segmentation, object detection, and more.

Hopefully there will be more topics in this section in the future. If you are interested in contributing, please check out the Contributing Guide.
